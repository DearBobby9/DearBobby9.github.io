Schoger Principles Applied — Change Log

#1 Ring > Border

Before: border border-border/50 on cards
After: ring-1 ring-zinc-950/[0.07] shadow-sm
Ring and shadow share the same render layer — no muddy band between them.

#3 Inset Ring

Before: dark:border dark:border-white/[0.18]
After: ring-1 ring-inset ring-zinc-950/[0.05]
5% opacity — barely visible, just enough to define container edges.

#6 Eyebrow Labels

Before: Section headers were just <h2>Projects</h2>
After: Added monospace eyebrow above: font-mono text-[11px] uppercase tracking-[0.15em]

#7 text-balance / text-pretty

Before: No text-wrap control anywhere
After: text-wrap: balance on headings, text-wrap: pretty on paragraphs. Eliminates orphan words.

#8 2x Line Height

Before: Hero subtitle leading-relaxed (1.625)
After: line-height: 2 (200%) on subtitle. More breathing room for short descriptive paragraphs.

#9 Split Layout Headers

Before: Section title centered or full-width alone
After: Title 3/5 left, description 2/5 right, top-aligned. Clear visual hierarchy like Tailwind's own site.

#10 Inline Heading

Before: "Now" title + rule line, no subtitle
After: <span bold>Now</span> <span muted>— what I'm working on</span> in one line. Compact, editorial.

#11 ch Units

Before: max-w-2xl (42rem fixed)
After: max-w-[58ch] scales with font size. Always optimal reading width.

#12 Pill Buttons

Before: rounded-md h-8 ghost buttons
After: rounded-full px-3 py-1.5 ring-1. Padding-based height, pill shape, subtle ring outline.

#14 Well Containers

Before: bg-muted on image containers
After: bg-zinc-950/[3%] ring-1 ring-inset ring-zinc-950/[0.04]. Subtle recessed depth without competing with images.

#16 Canvas Grid

Before: No section dividers
After: .canvas-grid::before adds full-width 1px line at section tops. .grid-lines::before/after adds vertical rules at container edges.

Removed: glass-panel

Before: .glass-panel on every section (blur, bg opacity, border)
After: Removed from sections. Glass panel adds visual noise on a white page. Sections should breathe, not be boxed. Keep glass only if over particle background with readability issues.
